# 💻 Gestion Financière — Frontend Web

Application web de gestion financière pour IDER SI.

## 🚀 Stack technique

- **Framework** : Next.js 16
- **Langage** : TypeScript
- **Style** : Tailwind CSS
- **Graphiques** : Recharts

## ⚙️ Installation

```bash
git clone https://github.com/Dahmoud4723/gestion-financiere-frontend.git
cd gestion-financiere-frontend
npm install
```

## ▶️ Lancement

```bash
npm run dev
```

Ouvrir http://localhost:3000

## 📱 Pages

| Route | Description |
|-------|-------------|
| /login | Connexion |
| /register | Inscription |
| /dashboard | Tableau de bord |
| /comptes | Gestion des comptes |
| /transactions | Liste des transactions |
| /categories | Gestion des catégories |
| /budgets | Budgets avec progression |
| /alertes | Alertes budget |
| /rapports | Rapports mensuels |
| /profil | Profil utilisateur |
| /admin | Panel administrateur |

## ✨ Fonctionnalités

- 🔐 Authentification JWT (USER/ADMIN)
- 📊 Dashboard avec 4 graphiques Recharts
- 💸 Gestion comptes, transactions, budgets
- 🔔 Alertes automatiques à 80% et 100%
- 📄 Export PDF et Excel
- 🌍 Support paiements locaux (Bankily, Masrvi, Sedad)
- 💱 Devise MRU
